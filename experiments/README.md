## Experiment 0: 基線對比（Phase 0 必做）

**問題**：Claude 規劃 + Qwen 執行 vs 單純 Claude，哪個更好？

**方法**：
- 用同一組任務（建議用 Louise v2 的 11 個優化）
- 兩種模式各跑一遍
- 記錄 token、時間、錯誤、最終質量

**輸出**：`baseline-vs-hybrid.md`

## Experiment 1-9: 三模式三類任務矩陣（Phase 3）

|  | 簡單任務 | 中等任務 | 複雜任務 |
|--|---------|---------|---------|
| **單 Claude** | A1 | A2 | A3 |
| **Claude + Qwen（手動）** | B1 | B2 | B3 |
| **gstack-plus（自動）** | C1 | C2 | C3 |

**測量指標：**
- Token 消耗（Tier-A vs Tier-Mid vs Tier-Exec 占比）
- 時間
- 錯誤次數
- 最終質量（用 `/review` 評分）
- 用戶介入次數
